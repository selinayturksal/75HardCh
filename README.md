# Seventyfive ⚡ [Work In Progress]

**Seventyfive** is a mobile application built with **Flutter**, currently under active development. It is designed to help users track their progress during the rigorous 75-day mental toughness challenge.

## 🛠 Project Status: Under Development
This project is a work in progress. I am currently focusing on core logic and UI implementation. New features are being added regularly.

## 🚀 Planned Features
* **Daily Task Checklist:** Track water intake, workouts, reading, and diet.
* **Progress Visualization:** Monitor your 75-day streak and history.
* **Photo Journal:** Securely store and view daily progress photos.
* **Reminders:** Custom notifications to ensure no task is forgotten.
